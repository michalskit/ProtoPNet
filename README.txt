# ProtoPNet (Fork)

This is a fork of the [ProtoPNet](https://github.com/cfchen-duke/ProtoPNet) project, which implements the prototypical part network described in the paper "This Looks Like That: Deep Learning for Interpretable Image Recognition" (NeurIPS 2019) by Chaofan Chen, Oscar Li, Chaofan Tao, Alina Jade Barnett, Jonathan Su, and Cynthia Rudin.

The original code package was developed by the authors at Duke University and licensed under the MIT License.

## Purpose of this Fork

The purpose of this fork is to reproduce the same augmentation process as in the original paper.
